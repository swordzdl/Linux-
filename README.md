## #chapter-01  虚拟机的安装和系统开发环境配置

### 虚拟机环境

虚拟机版本：VMWare16.0  安装流程省略

虚拟机配置：内存2G，2核CPU，20G硬盘，网络配置（桥接模式，自动）

网络配置流程：虚拟机安装过程中网络模式选择桥接模式，后期进入Ubuntu系统中将对应网络设置进行更改，除IP改为主机网段下外其余都与主机保持一样。ps：Windows11下查看网络IP需要进入  网络->高级网络设置->以太网
